Class Pair


This represents a key/value pair.

Example
var namePair = new Pair("name", "John Smith");
var agePair  = new Pair("age", 35);
alert("Pairs: " + [namePair, agePair]);


Constructor Arguments
Name Type Default Description
key Object   The key
value Object   The value

Properties
