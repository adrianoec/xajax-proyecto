Class StringMap

Extends Class: Map

A Map that gets its pairs using a single string.
The String has a pair separator and a name/value separator.
Ex: name1=value1&name2=value2&...

Example
var map = new StringMap("name-John Smith|age-35", "|", "-");
alert(map.getPairs());


Constructor Arguments
Name Type Default Description
string String   The String containing the pairs
nameSeparator String & The String that separates the pairs
valueSeparator String = The String that separates the name from value
isEncoded boolean true Determines if the String is HTML encoded

Properties
