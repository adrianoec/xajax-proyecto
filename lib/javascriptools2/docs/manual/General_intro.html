General

Introduction

JavaScripTools is a free, open source, set of JavaScript libraries, which should modern browsers (Internet Explorer 5.5+, Mozilla, Firefox, Konqueror and Opera). Note that on some browsers (Konqueror and Opera), some features (specially masks) may have less features than in Firefox and Internet Explorer. Please, help improving browser support by reporting compatibility issues to lfpg.dev at gmail dot com.

Currently, there are four scripts:

  • JavaScriptUtil: Common functions and classes
  • Parsers: A set of data parsers, allowing the parsing and formatting of objects like numbers and dates, which enables internationalization support at JavaScript level.
  • InputMask: Masks to format and restrict the user input on form fields.
  • JavaScripTable: A full featured dynamic table.
