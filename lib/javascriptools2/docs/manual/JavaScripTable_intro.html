JavaScripTable

Introduction to JavaScripTable

JavaScripTable is a dynamic and fully customizable table. It supports:

  • Two operation modes: client side or server side
  • Automatic paging, sorting and navigation
  • Full appearance control using CSS classes
  • Row selection (single or multiple)
  • Easy row and column manipulation
  • Cell edition, using customized controls or masks
  • And many more...
