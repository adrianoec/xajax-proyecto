JavaScripTools Samples

Here is a listing of some samples using the JavaScripTools scripts:

  • JavaScripTable
    • Simple client table
    • Rich client table
    • Server table
       
  • InputMask
    • InputMask, NumberMask, DateMask and SizeLimit
       
  • Parsers
    • NumberParser, DateParser and MapParser
       


Note: All regional formats on the samples are, by default, configured to brazilian locale. But it's easy to customize it to any other formats by changing some constants on the scripts. Check the API documentation for more information on wich constants may be changed, specially the Parsers and InputMask.
